# JailbreakedKindleGPT
chatGPT for jailbreaked Kindle

# Requirements
Kindle Paperwhite
USB Cable
openAI API Key

# Steps 
1. Jailbreak your Kindle
2. Install kTerm
3. Get openAI API Key
4. Edit settings in the .sh file
5. Run with bash kindlegpt.sh
